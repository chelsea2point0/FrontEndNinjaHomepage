Framework Page

For my Framework project, I created a webpage the provided a photo and a description of three fun adventures I've had with my husband. The header was centered on the page, and underneath there were 2 columns. The left side held the pictures and the right side held the descriptions, whith the text taking up more width than the photographs. I used CSS row and column classes to create this layout. This was also the first project where I used Google Fonts to create a custom look, exactly as I wanted it. I also used a media tag to change components of the CSS when the viewer's display size changed.

For this project I created a basic blogging webpage. The HTML contains anchor tags in an ordered list to allow the reader to jump to particular entries on the page. Unique ID tags were given to particular paragraphs to allow the page links to jump to the correct part of the page when clicked. The CSS also specifies font-family, background and text colors, font weight and text decoration to make the webpage more visually appealing.
